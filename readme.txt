== Description ==

Checkout Custom Fields Builder for WooCommerce is an extension, which enables users to easily customize the checkout fields and checkout page appearance. With Checkout Custom Fields Builder, you can add, remove, and edit any checkout fields, making the process simple and personalized.

# Who is This WooCoommerce Checkout Extension For?

Checkout Custom Fields Builder is perfect for WooCommerce online store & website owners who want to have more control over their checkout process. Whether you’re a small business owner or managing a large e-commerce platform, Checkout Custom Fields Builder for WooCommerce helps you create a custom checkout page with required fields for Shipping, Billing and Profile sections.

# What Can You Do with This WooCommerce Extension?

One of the primary capabilities of Checkout Custom Fields Builder for WooCommerce is the ability to disable or enable any checkout field. This feature allows you to tailor the checkout experience to your specific needs, ensuring that customers only see the fields that are relevant to them. For instance, if you don't need a company name or address field, you can easily turn it off.

Adding custom fields is another powerful feature of Checkout Custom Fields Builder for WooCommerce. You can create various types of fields to collect the necessary information from your customers:

- **Text Field**: For basic input.
- **Text Area:** For longer responses.
- **Number:** For numerical data.
- **Phone Number:** Ensures the correct format for phone numbers.
- **Email:** Ensures the correct format for email addresses.
- **Checkbox (only one option):** For yes/no or agree/disagree questions.

Checkout Custom Fields Builder for WooCommerce also allows you to delete or edit existing fields, providing the flexibility to refine your checkout form over time. The drag-and-drop functionality makes it easy to change the location of fields, enabling you to create a logical and user-friendly layout without any coding knowledge.

#### Editing Existing Fields

You can modify the existing fields in the checkout form. This means you can change the labels, placeholders, and descriptions to match your store’s needs. You can also decide if a field should be required or optional, and whether it should be visible to customers.

#### Adding New Fields

Adding new fields is simple with Checkout Custom Fields Builder for WooCommerce. You can drag new fields into the existing form and customize them as needed. The types of fields you can add include:

- **Text Field**
- **Textarea**
- **Number**
- **Phone Number**
- **Email**
- **Single Checkbox**

#### Customizing New Fields

When adding new fields, you can set:

- **Label:** The name of the field that customers will see.
- **Placeholder:** The text inside the field before the customer fills it out.
- **Description:** Additional information or instructions about the field.
- **Required:** Decide if the field must be filled out before completing the purchase.
- **Visible:** Choose if the field should be shown to customers.

# Styling a Checkout Form in Elementor

With Checkout Custom Fields Builder for WooCommerce, you can easily customize your checkout form using the Checkout Form widget in Elementor. This lets you design your checkout page exactly how you want it.

#### Customizing Field Layouts

You can choose how wide each field is on the checkout form:

- **1/1 (Full Width):** Takes up the whole row, good for long text fields.
- **1/2 (Half Width):** Takes up half the row, perfect for shorter fields like first and last names.
- **1/3 (One Third Width):** Takes up one-third of the row, great for small fields.
- **2/3 (Two Thirds Width):** Takes up two-thirds of the row, useful for fields that need more space.

This helps you create a neat and organized checkout form.

#### Customizing Buttons

You can change the text on the buttons to make it more interesting and match your store’s style. This can help encourage customers to finish their purchases.

#### Styling the Form and Fields

You can change how the form looks to match your website. You can adjust colors, fonts and paddings to make the checkout form look nice and professional.

== Installation ==

This section describes how to install the plugin and get it working.

#### Minimum Requirements

- WooCommerce 3.0 or greater
- WordPress 5.6 or greater

### Installation Process

1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate "Checkout Custom Fields Builder for WooCommerce" plugin.
4. Click "Install Now", then "Activate", and that's all.
